SkarGrid
Início Rápido

Skargrid é uma biblioteca moderna para tabelas interativas, com filtros avançados, busca acentuada, temas e muito mais.

Instalação via CDN

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/skargrid@1.0.0/dist/skargrid.css">
<script src="https://cdn.jsdelivr.net/npm/skargrid@1.0.0/dist/skargrid.min.js"></script>

Instalação via NPM

npm i skargrid

Exemplo Básico

<!-- HTML -->
  <div id="darkTable"></div>

  <!-- JS -->
  const darkData = [
    {id:1,nome:'Carlos',status:'Ativo',cidade:'SP',idade:30},
    {id:2,nome:'Fernanda',status:'Inativo',cidade:'RJ',idade:26}
  ];
  const darkCols = [
    {field:'id',title:'ID'},
    {field:'nome',title:'Nome'},
    {field:'status',title:'Status'},
    {field:'cidade',title:'Cidade'}
  ];
  new Skargrid('darkTable', { data: darkData, columns: darkCols, theme: 'dark' });

    { field: 'id', title: 'ID', sortable: true },
