SkarGrid Community 2.0 local dev build

Carrega diretamente de dist/skargrid.min.js e dist/skargrid.css — não usa CDN nem npm publicado. Rode npm run watch (rebuild automático a cada alteração em src/) e apenas atualize esta aba para ver o resultado.

1. Grid completo

sort, paginação, busca global, seleção, filtros por coluna (texto/select/número/data), configuração de colunas, exportação CSV e XLSX. A coluna "Status" usa um render que retorna um Node (badge colorido) — sempre tratado como seguro, sem precisar de allowUnsafeHtml. As colunas "ID" e "Nome" são frozen: true — role a tabela horizontalmente para ver as duas coladas à esquerda. O rodapé mostra agregações (footerAggregates): contagem de linhas e média de idade, recalculadas ao filtrar/pesquisar.

2. Virtualização

50.000 linhas, virtualization: true, sem paginação (a lib não combina os dois). Role a lista para validar que não há long tasks nem vazamento de nós.

3. Persistência de estado (persistState)

Ordene, filtre, selecione linhas ou mude de página aqui e clique em "recarregar (simula F5)" — o estado volta do localStorage via getState()/setState(), sem precisar reimplementar nada manualmente.
