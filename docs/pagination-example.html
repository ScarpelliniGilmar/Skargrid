SkarGrid Pagination Demo

Pagination Mode

This demo shows 1,000 rows with traditional pagination. All data is loaded at once, but only the current page is displayed.

Features: Search, sorting, selection, filters, and export work across all pages!

Use case: When you have moderate datasets (hundreds to thousands of rows) and want simple navigation.
